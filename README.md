# Übermensch

Landing page for Übermensch, an open-weight-first cognitive runtime.

Built with Svelte 5, Vite and TypeScript. Three languages (EN / DE / ES) and two copy modes.

## Development

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:5777`.

## Production build

```bash
npm run build
```

The static site is written to `dist/`. It deploys on Vercel with the Vite preset (build command `npm run build`, output directory `dist`) and needs no extra configuration.
